## Bienvenue

Bienvenue sur la documentation de l'API de communes-finder.fr, avant de commencer, rendez vous sur la page [Applications](https://communes-finder.fr/espace-client/applications) puis créez une application afin d'obternir des access tokens.

## Authentication
Tout les points d'entrés de l'api sont des points d'entrées sur lesquels vous devez être authentifié.
Chaque requête doit contenir un header `Authorization` avec le token d'authentification.

```json
"Authorization": "Bearer <token>"
```
